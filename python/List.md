## List

***

### 설명

<br>

 Python 알고리즘 문제를 풀거나 자료구조를 구현할 때 많이 쓰인다.  
 "배열" 이라고 하며 인덱스에 따라 순서대로 데이터가 들어간다. 
순서는 0부터 센다.
 
***

### 메소드

<br>

* ```test = []``` : 배열 만들기 
* ```test.append(k)``` : ```k``` 를 배열 안에 넣기 
* ```len(test)``` : 리스트의 총 길이 
* ```test[i]``` : ```i```번째 데이터를 뜻함 
* ```test.pop()``` : 리스트의 ```마지막``` 데이터를 반환
* ```test.pop(n)``` : 리스트의 ```n```번째 데이터를 반환
