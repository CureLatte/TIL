## 노드 서버의 장단점
* ### `장점`
  * 멀티 스레드 방식에 비해 컴퓨터 자원을 적게 사용함 
  * I/O 작업이 많으 ㄴ서버로 적합
  * 멀티 스레드 방식 보다 쉬움 
  * 웹 서버가 내장되어 있음
  * 자바스크립트를 사용함 >> 셰계에서 1위 
  * JSON 형식과 호환하기 쉬움
* ### `단점`
  * 싱글 스레드 CPU 코어를 하나만 사용함
  * CPU 작업이 많은 서버로는 부적함
  * 하나 뿐인 쓰레드가 멈추지 않도록 관리해야함 
  * 서버 규모가 커졌을 때 서버를 관리하기 어려움 
  * 어중간한 성능
