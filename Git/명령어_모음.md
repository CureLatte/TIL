## Git 명령어 모음

<br>

***
<br>

#### ```git init``` : 해당 폴더 및 프로젝트를 git으로 관리하기 윈한 파일 설치

<br>

#### ```git remote add <repositName> <url>```:```<url>``` 을  ```<repositName>```으로 저장 

<br>

#### ```git push <Name> <branch>``` : ```<name>``` 의 원격 저장소로 ```<branch>``` 를 생성하여 ```push```

<br>

#### ```git clone <url>``` : 해당 폴더에 ```<url>``` 저장소를 복제 (폴더가 만들어짐)

<br>
